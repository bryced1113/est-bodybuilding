## Design Elements & Icons

### Dumbbell Icon Specifications:
**Style:** Sharp, angular, geometric - NOT rounded or bubbly
**Design:** Rectangular weight plates with straight bar
**Usage:** Navigation logo, hero section, TRAIN pillar icon
**Color:** Black (or white on dark backgrounds)

**SVG Structure:**
- Left weight plate (rectangle)
- Left handle connection (small rectangle)
- Center bar (thin rectangle)
- Right handle connection (small rectangle)
- Right weight plate (rectangle)

---

## Animations & Interactions

### Hover Effects:
- **Buttons:** Darken slightly, smooth transition (300ms)
- **Product Cards:** Increase shadow, image scales up (500-700ms)
- **Navigation Links:** Darken text color (300ms)
- **Philosophy Cards:** Light gray background (500ms)
- **Size Buttons:** Black border appears (300ms)

### Scroll Effects:
- Smooth scrolling between sections
- Animated bouncing chevron in hero

### Responsive Behavior:
- **Mobile:** Stack all columns, hamburger menu, full-width buttons
- **Tablet:** Adjust column counts, maintain spacing
- **Desktop:** Full multi-column layouts, optimal spacing

---

## Shopify Implementation Notes

### Recommended Apps/Features:
1. **Custom Theme:** Use a minimal theme (Dawn, Debut, or custom)
2. **Page Builder:** Consider Shogun, PageFly, or GemPages for exact layout control
3. **Product Variants:** Set up size variants (M, L, XL, XXL)
4. **Inventory Management:** Enable "In Stock" badges
5. **Custom CSS:** Will need custom CSS for exact styling

### Key Shopify Settings:
- Enable size variants for products
- Set up product collections (if expanding product line)
- Configure shipping and taxes
- Set up payment gateways
- Add product images (awaiting custom photos)

### Custom Code Requirements:
- Custom navigation design
- Custom product card styling
- Philosophy section (could be a custom page/section)
- CTA section styling
- Footer customization
